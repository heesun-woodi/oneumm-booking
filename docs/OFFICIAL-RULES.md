# 온음 공동체 주택 놀터 이용 규칙 (공식)

> 출처: 온음 공식 구글 문서
> 분석일: 2026-03-04

---

## 📜 1장. 총칙

### 1조 (목적)
온음 공동체 주택 공용 공간 '놀터'를 **회원과 외부인이 조화롭고 원활하게 이용**할 수 있도록 대관 및 사용에 대한 명확한 기준과 절차 제시

### 2조 (용어의 정의)

#### 대관 (外部人)
- **외부인으로만** 구성된 모임
- 놀터를 제한된 시간 독점 이용

#### 사용 (會員)
- **회원(입주자)이 포함된** 모임
- 놀터를 제한된 시간 독점 이용

#### 회원
- 온음 공동체 주택의 **입주자**

---

## 🔄 2장. 이용 절차

### 1조 (대관 절차) - 외부인

1. **이용자** → 대관 담당자에게 문의
2. **대관 담당자** → 이용 가능 여부 확인
3. **대관 담당자** → 일정 확정 + 안내 문자 발송
4. **대관 담당자** → 놀터 캘린더에 등록

### 2조 (사용 절차) - 회원

1. **이용자** → 이용 가능 여부 직접 확인
2. **이용자** → 놀터 캘린더에 직접 등록

---

## 💰 3장. 비용

### 1조 (대관) - 외부인

- **기본**: 시간당 **14,000원**
- **장기 대관**: 전체 비용의 **70% 할인**
  - 예: 10시간 = 140,000원 → 42,000원 (98,000원 할인)

### 2조 (사용) - 회원

- **무료**: 가구당 **월 3회**
- **초과 시**: 대관과 동일 (시간당 14,000원)

---

## ⏰ 4장. 시간제한

### 1조 (대관) - 외부인

- **이용 가능 시간**: **09:00 ~ 23:00**
- **당일 대관**: ❌ **불가능**

### 2조 (사용) - 회원

- **시간제한**: ❌ **없음** (24시간 가능)

---

## 📋 5장. 이용 수칙

### 1조 (쓰레기 및 오물 처리)

- **원칙**: 각자 가지고 가기
- **정리정돈**: 필수

### 2조 (기타)

- **반려동물**: 동반 시 대관 담당자에게 **사전 고지 필수**
- **주차**: 대관 시 **불가**

## 💡 시스템 구현 시 반영 사항

### 대관 (외부인) 규칙
```javascript

  userType: "external",
  hourlyRate: 14000,
  longTermDiscount: 0.7,
  operatingHours: "09:00-23:00",
  advanceBooking: true, // 당일 불가
  minAdvanceHours: 24, // 최소 1일 전
  parking: false,
  petNotice: true
}
```

### 사용 (회원) 규칙
```javascript
{
  userType: "member",
  freePerMonth: 3,
  afterFreeRate: 14000,
  operatingHours: null, // 제한 없음
  advanceBooking: false, // 즉시 가능
  parking: false,
  petNotice: true
}
```

- 외부인은 담당자 통해서만
